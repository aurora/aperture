Tutorials

These tutorials are primarily aimed at developers who want to use one of more parts of Aperture. They should also be read by people that want to create custom implementations of Aperture's APIs.

The following topics concern everyone wanting to use one or more Aperture components:

  • General structure of Aperture components
  • RDF usage in Aperture

Furthermore, a number of component-specific tutorials are available. These tutorials are placed in the typical order of operation for applications that use Aperture to crawl an information source and extract and index all available information found in it.

Component-specific tutorials:

  • DataSources
  • DataAccessors
  • Crawlers
  • LinkExtractors
  • Security
  • MIME type identification
  • Extractors

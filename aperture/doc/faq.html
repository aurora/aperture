Frequently Asked Questions

Q: What source types does Aperture support?
A: Currently Aperture can crawl file systems (including shared network drives and removeable media), websites and IMAP mail boxes. An Outlook crawler is in the works.

Q: Which file formats does Aperture support?
A: See the page on Extractors for a list of file formats, their dependencies and remarks regarding their quality.
